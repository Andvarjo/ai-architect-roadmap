# Week 1 — LangGraph básico

**Dates:** 2026-04-27 → 2026-05-01
**Theme:** Bootstrapping LangGraph — graphs, tools, branching.

## Goal

End the week with a small but real LangGraph agent that:

1. Holds state across turns.
2. Calls at least one tool.
3. Branches on a condition (e.g. "do I need to call a tool, or answer directly?").

This is the *minimum* a senior engineer should be able to throw together in an afternoon. It's the starting point, not the destination.

## What's in the box (will fill in throughout the week)

- `01-basic-chatbot.py` — official tutorial chatbot.
- `02-with-tools.py` — same chatbot, now with a tool node + StateGraph.
- `03-branching.py` — conditional edges deciding the route.
